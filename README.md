## Functionalities
- [ ] User can create account
- [ ]  View other DSC's
- [ ]  Add your DSC
- [ ]  Update your DSC
- [ ]  Search by name
- [ ] Filter  by Location and Domains

## Instructions to run

* Pre-requisites:
	-  NodeJs

* Directions to setup/install
```bash
npm install
```

* Directions to execute

```bash
npm start
```
